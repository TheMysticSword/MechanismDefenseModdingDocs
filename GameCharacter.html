@:value([])customVars:Map<String, Dynamic> = []

A Map for storing custom variables.

Changing this Map does not send the values over network by default. For that, use the syncVar functions with "cv:<custom var name>" as the variableName argument.

Example:

var effectDuration = 0.5;
self.customVars.set("effectDuration", effectDuration);
self.syncVarFloat("cv:effectDuration", effectDuration);
