Getting Started

Mod folder structure:

  • _polymod_meta.json - contains information about your mod
  • LICENSE.TXT - licensing terms for your mod
  • _append/ - anything in this folder will append its content to the end of existing game files
  • _merge/ - anything in this folder will be merged with specific content in existing game files if the item key is matching. See Merging Files for examples.
  • Any other files in your mod's folder will be added to the game, or replace existing files if the paths are matching.

You can take a look at base game files in assets/data/ for examples on creating content.

The game comes with an example mod located at mods/example/.

Exposed functions

  • add(obj:FlxBasic) - add an object to the current state
  • globalTimeScaleManager - the current state's time scale manager
  • elapsedUnscaled - the amount of time in seconds passed since the last frame without the time scale manager's influence
  • play2DSound(options:Play2DSoundOptions) - play a sound in the world
  • createSoundPlayerObject(options:Play2DSoundOptions) - create a SoundPlayerObject that you can attach and network
  • spawnTempFX(options:TempFXSpawnOptions) - spawn a TempFX object
  • raycast(x:Float, y:Float, angle:Float, maxDistance:Float = 1000, withSolidTiles:Bool = true, withCharacters:Bool = false):RaycastResult - cast a ray that collides with the first object it meets
  • getColorFromString(str:String):Null<FlxColor> - alias for FlxColor.fromString
  • interpolateColors(Color1:FlxColor, Color2:FlxColor, Factor:Float = 0.5):FlxColor - alias for FlxColor.interpolate
  • loadScript(path:String, parseImmediately:Bool = false):Script - load a custom script from assets/data/scripts/
  • runGlobalScript(name:String, ?extraVariables:Map<String, Dynamic>):Dynamic - run a global script from assets/data/scripts/global/ (without the .hscript extension)
  • getGameValue(name:String):Dynamic - get a game value defined in assets/data/gameValues.csv. Returns null if the value is undefined.
  • editGameValue(name:String, newValue:Dynamic) - edit a game value at runtime
  • getModifiedStat(name:String, currentValue:Float, ?extraVariables:Map<String, Dynamic>):Float - apply a stat modifier from assets/data/scripts/statmods/ to an input value and return the modified value. If the statmod file doesn't exist, the value is unchanged.
  • getSpriteImage(path:String):Null<SpriteImage> - get a SpriteImage from path defined by sprites.csv
  • replaceFlags(string:String, flags:Array<String>, values:Array<String>):String - alias for firetongue.Replace.flags
  • getPoint(x:Float = 0, y:Float = 0):FlxPoint - get a temporary FlxPoint from a pool. Make sure you call point.put() after you're done using the point to put it back in the pool.
  • Only during a game:
    • shakeCamera(power:Float, ?x:Float, ?y:Float, ?radius:Float) - shake the camera, optionally making the shaking weaker depending on the distance from the shake source
    • kickCamera(power:Float, direction:Float) - kick the camera in a certain direction
    • givePlayersScrap(amount:Float) - give scrap to all players
    • triggerGameOverServer() - start the game over sequence. Does nothing if executed on a non-server.

Exposed variables

  • serverActive:Bool - true if the local machine is hosting an online game, or if the game is local
  • clientConnected:Bool - true if the local machine is connected to an online game, or if the game is local
  • Only during a game:
    • revivalMechanism:GameCharacter - the Revival Mechanism instance
    • levelVariant:LevelVariant - the current level variant
    • wave:Int - the current wave
    • waveActive:Bool - true if we are currently fighting a wave
